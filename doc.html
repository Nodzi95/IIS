Implementace

Veškeré vykreslování html kontrukcí se provádí v souboru stranky.php voláním různých funkcí. Většina dotazů na databázi se provádí v souboru select.php, některé (např. pro zobrazování dat z databáze na stránku) jsou implementovány v souboru stranky.php.

Úvodní menu

Úvodní menu je řešeno voláním funkce function main, která vykresluje stránky, pokud není nikdo přihlášen. Zde se dá přihlásit do systému, registrovat nebo zjistit informace o firmě.

Admin

Pokud se přihlásí administrátor (který vznikne pouze tak, že se "natvrdo přidá" do databáze) je volána funkce function admin, kde je implementován případ užití pro vytvoření a smazání zaměstnance, gate, letadla a terminálu. Navíc je umožněno administrátorovi, aby si editovat svůj profil (jméno, příjmení, login a heslo). Z diagramu případu užití byly pro administrátora vynechány tyto možnosti: editovat gate, terminal, letadlo a zrevidovat letadlo, jelikož by to implementačně neznamenalo nic nového, jen opakování stejného postupu jak u editování profilu.

Zaměstnanec

Při přihlášení jako zaměstnanec (zaměstnanec vznikne jen tak, že ho vytvoří administrátor) se volá funkce function zam, kde jsou implementovány připady užití vytvoření letu, smazání letu a stejně jak administrátor si může editovat/smazat profil. Zaměstnanec může plánovat lety jen do budoucnosti (zítra a dál).

Zákazník

Jako poslední se jde přihlásit jako zákazník (vznikne registrací v úvodním menu), zde si může zákazník objednat nebo stornovat letenky, dále si může editovat/smazat účet. Uživatel může stornovat jen letenky na let, který se uskuteční v budoucnosti (zítra a dál).

Instalace

  • Na server není třeba nic instalovat
  • Informační systém byl vyvíjen a testován na PHP 5.6.25 (školní server eva) a prohlížeči Chrome
  • Script by se měl sám přihlásit k existující databázi
